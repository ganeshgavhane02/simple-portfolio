# Portfolio

This is a simple personal portfolio website built using HTML and CSS.

## About

I am Ganesh D Gavhane, and this project shows my journey in learning web development. I am currently practicing:

- HTML structure and semantic markup
- CSS styling and layout
- Responsive design basics
- Simple portfolio presentation

## Features

- Clean single-page portfolio layout
- About, skills, projects, and contact sections
- Image assets for profile and projects
- Responsive form styling and readable sections

## Files

- `portfolio.html` - main portfolio page
- `stle.css` - stylesheet for the portfolio
- `images/` - SVG image assets used on the page

## Preview

Open `portfolio.html` in your browser to view the site.
